Setting up a new Site with Wordpress and Semplice

Step Options
1 Domain & Hosting. I recommend Dreamhost, which is ~$10/month or less, with a free domain name. They also provide a nice one-click install of Wordpress.
Here's a coupon for 25% off.
Once you get through the registration and purchase process, Dreamhost will confirm that your domain is ready, and you can go to your new domainname.com and see a temporary 'This Site is Coming Soon' screen.
2 Install Wordpress. If you're using Dreamhost, Then look for Goodies > One Click Installs. Select Wordpress, and in the options that pop up, select:
  • Install to: Select a Domain (pick yours here from the dropdown list): / and leave it blank after the slash.
  • Select Database: Automatically Create Database
  • Deluxe Install? Leave it checked.
Some other web hosts have a handy automatic install of Wordpress too - they will look and work differently but are probably fine.
3 Create your Admin username and password / follow the prompts in email.
4 Download Semplice, a templating layer on top of Wordpress. You'll have to buy it, then download it. Unzip the file, and inside will be some instructions an another zip file called semplice4_single.zip. DON'T unzip that one - just leave it for now.
5 Install the Semplice theme. Log into your Wordpress installation, look in the left hand menu for Appearance > Themes. Or you can go directly to http://yoursite.com/wp-admin/theme-install.php.
Click 'Add a new theme', then 'Upload a theme' and upload the zip file called semplice4_single.zip.
After the theme uploads, click 'Activate.'
6 That's the hard part! Now you can use the Semplice control layer to make pages. Semplice has great help tutorials here.
